Web technologies

html

(HypterText Markup Language) is the fundamental building block of the Web and defines the content and structure of a web page. Below are some common page elements that HTML is used for:

  • Headings
  • Paragraphs
  • Images
  • Links
  • Lists

Learn more about HTML with the w3Schools HTML Tutorial.

CSS

(Cascading Style Sheets) is used to describe a web page's appearance and presentation. Below are some common uses of CSS:

  • Fonts
  • Colors
  • Layout
  • Backgrounds

Learn more about CSS with the w3Schools CSS Tutorial.

JavaScript

(JAVASCRIP JavaScript) is the programming language of the web and controls the functionality and behaviour of a web page. Below are some common uses of JavaScript:

  • Add Interactivity to a Web Page
  • Create Web Apps
  • Create Web Games& Animations

A great JavaScript reference isThe Modern JavaScript Tutorial.
